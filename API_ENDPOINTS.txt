

##  Authentication & Profile 

POST    /api/register           تسجيل مستخدم جديد
POST    /api/login             تسجيل الدخول للحصول على رمز المصادقة
POST    /api/logout            تسجيل الخروج
POST    /api/profile/photo     رفع صورة الملف الشخصي

##  Events Management 

### Public Event Routes 
GET     /api/events            عرض جميع الفعاليات المتاحة للجمهور
GET     /api/events/{id}       عرض تفاصيل فعالية معينة
GET     /api/events/{id}/program    عرض جدول فعاليلة معينة
GET     /api/events/{id}/workshops   عرض جدول ورشة معينة
POST    /api/sessions/{id}/questions    ask a question
DELETE  /api/questions/{id}    delete a question (role: event_organizer can delete any question)
GET     /api/sessions/{id}/questions    show all questions and their corresponding answers
POST    /api/questions/{id}/upvote      upvote a question, if called again the upvote is toggled (like/unlike)
POST    /api/questions/{id}/answers      answer a qustion
DELETE  /api/answers/{id}    delete an answer (role: event_organizer can delete any answer)
POST    /api/sessions/{id}/survey    answer a survey


### Organizer Only Routes (للمنظمين فقط)

POST    /api/events            إنشاء فعالية جديدة (role: event_organizer)
PUT     /api/events/{id}       تحديث فعالية موجودة (role: event_organizer)
DELETE  /api/events/{id}       حذف فعالية (role: event_organizer)

POST    /api/events/{id}/sessions   انشاء حصة في فعالية معينة (role: event_organizer)
PUT     /api/events/{id}/sessions/{id}    تحديث حصة معينة في فعالية معينة (role: event_organizer)
DELETE  /api/events/{id}/sessions/{id}    حذف حصة معينة في فعالية معينة (role: event_organizer)

POST    /api/events/{id}/sessions/{id}/assign-submission    تعيين عرض تقديمي/مقترح إلى حصة (role: event_organizer)
PATCH   /api/events/{id}/sessions/{id}/presentations/{id}/time     تعيين وقت البدء/النهاية لمقترح مقبول و مخصص (role: event_organizer)

POST    /api/events/{id}/periods   انشاء فترة في فعالية معينة (role: event_organizer)
PUT     /api/events/{id}/periods/{id}    تحديث فترة معينة في فعالية معينة (role: event_organizer)
DELETE  /api/events/{id}/periods/{id}    حذف فترة معينة في فعالية معينة (role: event_organizer)

POST    /api/events/{id}/workshops   انشاء ورشة في فعالية معينة (role: event_organizer)
PUT     /api/events/{id}/workshops/{id}    تحديث ورشة معينة في فعالية معينة (role: event_organizer)
DELETE  /api/events/{id}/workshops/{id}    حذف ورشة معينة في فعالية معينة (role: event_organizer)

GET     /api/sessions/1/survey-results     get the results of a survey after it ends(3 minutes after the session ends) (role: event_organizer)

##  Submissions Management 

### For Authors & Committee 
GET     /api/submissions/my    عرض مقترحات المستخدم الحالي
POST    /api/submissions       إنشاء مقترح جديد (رفع PDF + بيانات)
GET     /api/submissions/{id}  عرض تفاصيل مقترح معين
PUT     /api/submissions/{id}  تحديث مقترح موجود (للمؤلف فقط، في حالة pending)
DELETE  /api/submissions/{id}  حذف مقترح (للمؤلف فقط، في حالة pending)

### For Organizers & Scientific Committee (للمنظمين واللجنة العلمية)
GET     /api/submissions       عرض جميع المقترحات مع إمكانية الفلترة
POST    /api/submissions/{id}/status  تحديث حالة المقترح (قبول/رفض/مراجعة)

---

##  Workshops Management

### For Workshop Facilitator (role: workshop_facilitator)
GET    /api/workshops/{id}/submissions    عرض تسجيلات ورشة معينة
POST   /api/workshops/{id}/submissions/{id}/moderate     قبول او رفض تسجيل معين في ورشة معينة
GET    /api/workshops/{id}/participants    عرض المستخدمين المقبولين في ورشة معينة
POST   /api/workshops/{id}/materials     رفع موارد لورشة معينة
DELETE /api/workshops/{id}/materials/{id}    حذف موارد معينة من ورشة معينة

### For Workshop Facilitators and registered Participants  (role: workshop_facilitator , participant)
GET    /api/workshops/{id}/materials     عرض كل الموارد لورشة معينة
GET    /api/workshops/{id}/materials     عرض/تحميل موارد معينة لورشة معينة

### For Participants
POST   /api/workshops/{id}/register    التسجيل في ورشة معينة
DELETE /api/workshops/{id}/unregister    الغاء التسجيل من ورشة معينة / حذف مسؤول ورشة معينة لمستخدم معين

### Event Creation (إنشاء فعالية)
```
title: مطلوب - عنوان الفعالية
description: مطلوب - وصف الفعالية  
start_date: مطلوب - تاريخ البداية (بعد اليوم)
end_date: مطلوب - تاريخ النهاية (بعد تاريخ البداية)
location: مطلوب - مكان الفعالية
theme: مطلوب - موضوع الفعالية
type: مطلوب - نوع الفعالية (congress, seminar, workshop, conference)
contact_email: مطلوب - بريد التواصل
contact_phone: اختياري - رقم الهاتف
scientific_committee: اختياري - أعضاء اللجنة العلمية (JSON)
invited_speakers: اختياري - المحاضرون المدعوون (JSON)
```

### Submission Creation (إنشاء مقترح)
```
event_id: مطلوب - معرف الفعالية
title: مطلوب - عنوان المقترح
authors: مطلوب - قائمة المؤلفين (array)
abstract: مطلوب - الملخص (أكثر من 100 حرف)
keywords: مطلوب - الكلمات المفتاحية (3 كلمات على الأقل)
type: مطلوب - نوع العرض (oral, poster, affiche)
pdf_file: اختياري - ملف PDF (أقل من 5 ميجابايت)
```

### Session Creation (إنشاء حصة)
```
event_id: مطلوب - معرف الفعالية
title: مطلوب - عنوان الحصة
room: مطلوب - غرفة الحصة 
start_time: مطلوب - وقت بداية الحصة (YYYY-MM-DD 00:00:00)
end_time: مطلوب - وقت نهاية الحصة (YYYY-MM-DD 00:00:00)
session_chair_id: مطلوب - معرف مسؤول الحصة
```

### Workshop Creation (إنشاء ورشة)
```
event_id: مطلوب - معرف الفعالية
title: مطلوب - عنوان الورشة
room: مطلوب - غرفة الورشة 
start_time: مطلوب - وقت بداية الورشة (YYYY-MM-DD 00:00:00)
end_time: مطلوب - وقت نهاية الورشة (YYYY-MM-DD 00:00:00)
animator_id: مطلوب - معرف مسؤول الورشة
description: اختياري - تعريف الورشة 
```

### Submission Assignment (تكليف مقترح/عرض تقديمي بحصة)
```
submission_id:  مطلوب - معرف الاقتراح
```

### Period Creation (إنشاء فترة)
```
event_id: مطلوب - معرف الحصة
title: مطلوب - عنوان الفترة
start_time: مطلوب - وقت بداية الفترة (YYYY-MM-DD 00:00:00)
end_time: مطلوب - وقت نهاية الفترة (YYYY-MM-DD 00:00:00)
room: اختياري - غرفة الفترة 
description: اختياري - شرح الفترة 
```


### Submissions Filtering (فلترة المقترحات)
```
event_id: فلترة حسب الفعالية
status: فلترة حسب الحالة (pending, accepted, rejected, revision)
```

### Status Update (تحديث الحالة)
```
status: مطلوب - الحالة الجديدة (pending, accepted, rejected, revision)
admin_notes: اختياري - ملاحظات إدارية
```

##  Authentication Requirements 

### Middleware Groups 
```
auth:sanctum - يتطلب تسجيل دخول صحيح
role:event_organizer - يتطلب دور منظم الفعاليات
role:author - يتطلب دور المؤلف
role:scientific_committee - يتطلب دور عضو اللجنة العلمية
role:author,scientific_committee,event_organizer - يقبل أي من هذه الأدوار
role:event_organizer,scientific_committee - يقبل المنظمين أو اللجنة العلمية
```

### Headers Required 
```
Authorization: Bearer {your_token}
Content-Type: application/json
Accept: application/json
```

##  File Upload Endpoints (رفع الملفات)

##  Response Format 

### Success Response 
```json
{
    "success": true,
    "message": "رسالة النجاح",
    "data": { }
}
```

### Error Response
```json
{
    "success": false,
    "message": "رسالة الخطأ",
    "errors": { }
}
```

## Frontend Implementation Notes 

### Role-based Navigation 
```
event_organizer: يصل لجميع إدارة الفعاليات + المقترحات
author: يصل للمقترحات الخاصة به + التقديم لفعاليات
scientific_committee: يصل لتقييم المقترحات + إدارة اللجنة
participant: يصل لعرض الفعاليات + التسجيل (قريباً)
```

- عرض progress bar للرفع
